# 실험 결과 보고서 (Experimental Result Report)
## Tier 1 — Ridge Linear Regression Baseline (Covariates 원복 재실행)

---

## 1. 실험 개요
- **대상:** 14개 시계열 통합 데이터셋 중 12개 처리 완료 (Shanghai T1DM/T2DM은 연속 레코드 부족으로 제외)
- **모형:** Ridge Regression (L2 alpha=1.0)
- **윈도우:** Lookback=6 steps, Horizon=6 steps
- **분할:** 환자별 연대기적 80% Train / 20% Test
- **총 처리 윈도우:** 약 4,992만 개
- **업데이트 사항:** AIDET1D, GLAM, IOBP2 등에서 손실(Drop)되었던 이벤트 기반 공변량(식사 시간 발생 여부, 인슐린 투여량 등)을 파일 전체 파싱 기반으로 완벽히 복원한 후 재실험을 진행했습니다.

---

## 2. 핵심 발견 사항

### 2-1. 단변량(Univariate) 성능 분포

| 성능 구간 | 데이터셋 | 특성 |
|---|---|---|
| **RMSE < 5** (매우 우수) | CGMacros_Libre (1.25), CGMacros_Dexcom (3.24) | 비당뇨인, 15분 간격, 풍부한 영양소 메타데이터 |
| **RMSE 10~20** (보통) | BIGIDEAs (14.37), CGMND (14.40), GLAM (13.87), HUPA-UCM (16.11), UCHTT1DM (18.49), Park_2025 (19.99) | 혼합 대상, 다양한 측정 주기 |
| **RMSE > 20** (높은 오차) | AIDET1D (23.63), IOBP2 (26.08), Bris-T1D_Open (29.56), PEDAP (30.47) | T1DM 환자군, 혈당 변동성 고극심 |

**해석:** T1DM 환자군(PEDAP, Bris-T1D_Open, IOBP2)의 혈당 예측 난이도가 비당뇨인(CGMacros)과 비교해 10~25배 높은 오차율을 보였습니다. 

### 2-2. 다변량(Multivariate) 이득(Delta) 분석 (복원 완료 데이터 기준)

| 개선 확인 | 데이터셋 | Covariates | Delta RMSE | 비고 |
|---|---|---|---|---|
| **향상** | CGMacros_Dexcom | 15개 | **+0.24** | 다수의 영양소 변동이 선형 규칙에 기여 |
| **향상** | BIGIDEAs | 2개 | **+0.08** | 칼로리 등 |
| **향상** | HUPA-UCM | 6개 | **+0.07** |  |
| **향상** | PEDAP | 2개 | **+0.04** | 기저 인슐린, 식사 투입으로 오차 0.04 절대적 감소 확인 |
| **향상** | GLAM | 1개 | **+0.01** | 식사 여부(EventMarker) 투입만으로 미세 감소 확인 |
| **무의미/동일** | CGMND, IOBP2, AIDET1D 등 | 0~6개 | 0.00 | 선형 관계로 설명 불가 (비선형 모형 필요) |

**핵심 관찰:**
1. **이벤트 공변량의 선형 회귀 한계:** GLAM과 PEDAP 등에서 다변량 공변량을 복원하여 투입했음에도 불구하고, Delta RMSE 향상 폭이 0.01~0.04로 매우 미미했습니다. 이는 "식사를 했다" 혹은 "인슐린이 들어왔다"는 사실 자체가 선형(Linear) 공식 안에서는 극적인 성과를 내지 못하고 있다는 뜻입니다. 

---

## 3. 결론: "왜 Tier 2 (Decision Tree/Random Forest)가 즉시 필요한가?"

- 선형 모형인 Ridge는 **"인슐린 5U가 투여되었을 때, 15분 뒤엔 급격히 감소하지만, 그 감소 양상은 이전 혈당이 높을 때와 낮을 때마다 완전히 다르다"**라는 조건부 결합(Interaction) 논리를 전혀 학습할 수 없습니다.
- 현재 공변량을 성공적으로 추출해주었음에도 오차가 크게 줄지 않는 현상은, 데이터의 질이 나빠서가 아니라 **모델의 구조가 혈당 메커니즘을 제대로 담지 못해서** 발생한 것입니다.
